# SFK Cheat Panel — BepInEx plugin for StrikeForce Kitty

## Installation
1. Install BepInEx 5 (win_x64) in the game directory
2. Copy `SFKMod.dll` to `BepInEx/plugins/`
3. Launch game, press **F1** to toggle panel

## Features

### Battle
- **God Mode** — cats take zero damage
- **God Mode (enemies+bosses)** — enemies/bosses take zero damage
- **Auto Heal** — auto-heal cats every 0.3s
- **Heal Now** — instant full heal
- **Kill All** — kill all enemies + bosses
- **Trap Immune** — ignore all traps

### Economy
- **+1000 / +10000 Fish** — add fish
- **Auto Fish 99999** — keep fish at max in levels
- **+1000 Gold** — add gold

### Speed
- **Speedhack** — toggle + slider (1x–10x)

### Unlocks
- **Unlock All Levels** — all 30 levels + bonus + 5 bosses
- **Unlock Costumes** — all 350 dress types
- **Unlock Artifacts** — all 29 artifacts + open all chests
- **Unlock Abilities** — max all 12 abilities + 15 skills
- **Unlock Teams** — all 105 teams with presets
- **Open Chests** — mark all 101 chests opened
- **UNLOCK ALL** — everything at once

## Build
```
bash build.sh
```
Requires mcs (Mono C# compiler).
